# BetSim - betting simulator v 1.0.0
# Based on sports-skills, using the real data. No API keys required.

Usage: 
1 - View matches — see all matches for today with IDs and odds
2 - Place a bet — enter event ID, choose outcome, enter amount
3 - Check status — use 'check' command to see if match finished
4 - Exit and reopen — program automatically settles pending bets
